# Installation

## Windows

To install Nokiatis Launcher for windows you just have to head over to our landing page and press download.
You'll then download the installer executable, which you can install.

Nokiatis Launcher is installed for your windows user only.

## MacOS

To install Nokiatis Launcher for MacOS you just have to head over to our landing page and press download.
You'll then download the macOS installer, which you can install.

The app is universal, which means it runs natively on both Intel and Apple Silicon.

## Linux

To install Nokiatis Launcher for linux you just have to head over to our landing page and press download.
You'll then download the AppImage and run it normally.
